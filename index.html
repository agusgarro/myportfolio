Myberlinmap

Myberlinmap was the second project I made for my web dev Bootcamp at Ironhack. We used express for the backend, Javascript HTML and Bootstrap for the frontend. We also used Mapbox. Is a map app where you can find restaurants near you in Berlin, and also write reviews that will be saved in your own profile for you to remember how you liked or disliked that restaurant.

e-meet

e-meet was the third project that I created in my life. We used Express for backend, react for frontend and bootstrap for styling. E-meet is an online platform for people to plan events in these corona times when it's so difficult to meet with people personally. It's connected to google meet and it has log in and sign up options.
